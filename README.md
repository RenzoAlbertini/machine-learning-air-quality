# Air Quality Prediction and Classification using Machine Learning

End-to-end Machine Learning project focused on air quality prediction and classification using real-world environmental data collected across Taiwan.

## Project Overview

This project explores the complete supervised Machine Learning workflow, from data preprocessing and feature engineering to model training, evaluation and interpretation.

The objective is to analyze air pollution indicators and develop predictive models capable of estimating air quality conditions using environmental measurements.

## Dataset

Taiwan Air Quality Index Dataset (2016–2024)

Features include:

- PM2.5
- PM10
- O3
- CO
- NO2
- SO2
- Wind speed and direction
- Geographical coordinates
- Temporal information

The dataset contains millions of environmental observations collected over multiple years.

## Machine Learning Pipeline

### Data Preparation

- Missing value handling
- Feature selection
- Feature engineering
- Categorical encoding
- AQI discretization
- Correlation analysis

### Exploratory Data Analysis

- Statistical summaries
- Correlation heatmaps
- Scatter plots
- Feature relationship analysis

### Models Implemented

#### Decision Tree Classification

Predict air quality status categories from environmental measurements.

#### Linear Regression

Estimate Air Quality Index (AQI) values from pollutant concentrations.

#### Polynomial Regression

Model non-linear relationships between air pollution indicators and AQI.
